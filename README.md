# Song BPM Analyzer & Waveform Visualizer using Arduino & Processing
### Components
- Arduino Uno / Arduino Nano ATmega328P
- Sound sensor module KY-038
- 8 LEDs
- 8 1k Ohm resistors
- Jumper wires
- Breadboard

### Software
- Processing 4.0.1
- Arduino 1.8.19
